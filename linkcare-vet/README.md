# LINKARE VET - Ferramenta de Análise Colaborativa

## Sobre

Ferramenta interativa para coletar e organizar sugestões da cliente Laura sobre o sistema LINKARE VET.

## Funcionalidades

- ✅ 11 seções de análise com prioridades definidas
- ✅ Suporte a Markdown (negrito, itálico, listas, links)
- ✅ Upload de imagens (converte para Base64)
- ✅ Pré-visualização ao vivo do texto formatado
- ✅ Controle de status por seção
- ✅ Histórico de ações com timestamp
- ✅ Exportação em Markdown para compartilhamento

## Como usar

1. Clique nas seções para expandir
2. Preencha observações usando Markdown
3. Faça upload de imagens se necessário
4. Altere o status de cada item
5. Clique em "Copiar Tudo em Markdown" para enviar feedback
